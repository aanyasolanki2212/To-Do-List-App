# To-Do List App

A Python-based task management application that allows users to add, view, and delete tasks through a menu-driven interface.

## Features
- Add Tasks
- View Tasks
- Delete Tasks
- Task Storage Using Text File
- Menu-Driven Interface

## Technology Used
- Python

## Concepts Used
- Lists
- Loops
- Conditional Statements
- File Handling
- User Input Handling
